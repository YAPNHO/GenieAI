## 网页链接
https://genie-2041352088626315264.sg.agentos-app.run

## 产品概述
这是一个基于人脸识别技术的智能签到系统，区分老师端和学生端，支持人脸注册与签到功能。老师端可以选择开启倒计时签到功能，学生应该在规定时间内完成签到，否则记为迟到。

## 核心功能
### 角色
首页提供老师端/学生端入口选择
### 学生端
人脸注册：输入姓名和学号，调用摄像头拍照注册人脸
人脸签到：输入学号，调用摄像头拍照，与注册照片进行人脸对比验证签到
### 老师端
签到记录查看：查看所有学生的签到记录，包含姓名、学号、签到时间
实时刷新：支持手动刷新签到列表
